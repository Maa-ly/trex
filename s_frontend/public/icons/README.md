# Icon Files

Save your generated PNG icons here with these exact names:

- `icon-16.png` - 16x16 pixels (Chrome UI)
- `icon-32.png` - 32x32 pixels (Chrome UI)
- `icon-48.png` - 48x48 pixels (Extension management)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## How to Generate:

Open `generate-icons.html` in a browser, then right-click each icon to save.

## Files in this folder:

- `icon.svg` - Master SVG source (can be edited)
- `generate-icons.html` - Icon generator tool (open in browser)
- `icon-*.png` - Generated PNG files (save these here)
